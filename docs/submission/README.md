## 仓库说明

本仓库为作品主体任务代码开源仓库，`main` 分支包含：

- ELF2 开发板端侧视觉、语音、对话与控制服务代码；
- PC / 移动端 Web 交互层与小程序侧代码；
- Cloudflare 公网接入网关；
- 板端 systemd 服务、一键启动与部署校验脚本；
- 核心测试与项目文档。

实际系统采用“ELF2 端侧推理与服务编排 + 公网 Web/小程序交互入口”的架构。开发板承担视觉感知、语音识别/合成、控制网关和状态汇聚等主链路任务，网页与小程序负责跨端呈现和交互输入。
